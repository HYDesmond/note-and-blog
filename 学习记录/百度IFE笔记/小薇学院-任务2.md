# 任务介绍（[传送门](http://ife.baidu.com/course/detail/id/92)）
制作一个带样式的html网页，在任务一的基础上为元素增加样式，可以自由发挥使用一些伪类做出更好看的效果。
# 需要知识点
+ html标签基本知识
+ 可以使用HTML5的语义化标签
+ CSS样式基本知识
+ [`box-shadow`属性](http://www.runoob.com/cssref/css3-pr-box-shadow.html)，为边框加上阴影效果
